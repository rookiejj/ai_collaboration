# Changelog

## [1.0.0] - 2026-03-24

### 추가
- 토론 모드: 분석가 / 비전가 / 현실주의자 / 악마의 변호인 4인 토론
- 협업 모드: 기획자 / 개발자 / 디자이너 / 비즈니스 전략가 전문가 협업
- 헤더 토글로 토론 ↔ 협업 모드 즉시 전환
- Claude / GPT-4o / Gemini 모델 슬롯별 선택 기능
- API 키 브라우저 localStorage 자동 저장
- 토론 중단 버튼 (오버레이 팝업 포함)
- 진행자 자동 클로징 멘트 (Claude API 기반)
- 로컬 CORS 프록시 서버 (OpenAI · Gemini 브라우저 직접 호출 우회)
- 우측 가이드 패널 (시작 방법, API 키 발급, 오류 해결)
- 역할별 고정 색상 (모델 변경 시에도 역할 색상 유지)
- 프록시 서버 연결 상태 실시간 표시
