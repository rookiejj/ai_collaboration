GPT-4o / Gemini 사용 시 — 프록시 서버 필수, 각각 별도 API 키 필요.
💡 추천 시작 방법
비전가·현실주의자 슬롯을 모두 Claude로 설정. Claude 키 하나만으로 4명이 토론.

자주 묻는 오류

Failed to fetch
프록시 서버가 실행되지 않았거나 포트가 다름. node proxy-server.js 재실행.
quota exceeded
API 크레딧 소진 또는 무료 한도 초과. 해당 서비스 결제 페이지에서 충전.
API key not valid
키 앞뒤 공백 확인 후 재입력. 새로 발급한 키는 1~2분 후 활성화.
